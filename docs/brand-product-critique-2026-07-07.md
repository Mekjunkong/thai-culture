# Brand + Product Critique — Thai Lessons Chiang Mai

Date: 2026-07-07
Scope: Logo, website brand system, and onsite Chiang Mai workbook/PDF product.

## Quick verdict

The product direction is strong: practical Thai missions for real Chiang Mai life. But the brand and PDF still need to feel more like a sellable learning product, not a temporary classroom handout.

## What is working

- Clear niche: Thai for expats and travelers in Chiang Mai.
- Strong promise: speak useful Thai in real situations, not textbook Thai.
- Good product mechanic: short missions, phrase builders, quizzes, and human correction.
- Warm visual direction: cream, indigo/navy, turmeric/gold, and temple red already feel Thai without being too touristy.

## Problems to fix before designing

### 1. No real logo system

Current header uses a simple `ไทย` badge. It is readable, but it does not create a memorable brand. It also does not scale into favicon, PDF cover, social preview, worksheets, or printable products.

**Fix:** create a full logo system:
- primary horizontal logo
- square icon / favicon
- PDF cover mark
- social preview graphic
- reusable React component

### 2. Brand mark needs to communicate both Thai + Chiang Mai

A random Thai-script badge says “Thai,” but not “Chiang Mai” or “real-life lessons.”

**Fix:** use a mark combining:
- Thai script `ไทย`
- Doi Suthep / mountain line for Chiang Mai
- sun/lantern/gold accent for warmth
- notebook/mission-card feeling through rounded product shapes

### 3. PDF cover is better than before, but still too plain

The workbook has useful pages, but visually it still feels like a generated handout. It lacks a strong cover, brand mark, premium section dividers, and real navigation.

**Fix:** add:
- branded logo on cover and footers
- stronger title hierarchy
- QR/link cards to online missions
- “what you get” product value box
- teacher correction rubric
- lesson package upsell page
- certificate / completion page

### 4. Product funnel is not obvious enough inside the PDF

The PDF should not only teach. It should also sell the next step naturally: book a trial, send a voice note, or buy Starter Pack.

**Fix:** each mission should end with:
- one real-world task
- one voice-note homework
- one QR/link to online mission
- one teacher correction box

### 5. PDF needs more classroom usability

For onsite Chiang Mai teaching, Mike needs pages he can actually use in a cafe/market lesson: quick warmup, roleplay, correction notes, and homework.

**Fix:** add teacher-facing tools:
- 60-minute lesson plan
- correction rubric: tone, rhythm, polite ending, confidence
- student tracker
- lesson report template
- 5-lesson package outline

## Design direction chosen

**Brand idea:** “Real-life Thai missions in Chiang Mai.”

**Logo concept:** rounded square mark with Thai word `ไทย`, Doi Suthep-style mountain line, gold sun, and warm cream/navy/turmeric colors. It should look friendly, practical, and printable.

**Product upgrade direction:** turn the PDF into a branded starter workbook + onsite lesson kit, not just phrase pages.
