How to use this workbook

For the student

Do not try to memorize everything. Choose one mission that matches your real life this week. Practice the phrase, use it once, then send a voice note for correction.

For the teacher

Teach one situation per lesson. Use the phrase builder, roleplay the likely reply, then walk the learner into a real cafe, market, lobby, or map situation.

Lesson promise

By the end of each lesson, the learner can say one useful sentence in a real Chiang Mai situation.

Pronunciation rule

Romanization is only a guide. First build confidence and politeness. Then correct rhythm, vowel length, and tone feeling with voice notes.

Foundation

Thai survival basics before missions

Polite endings

ครับ khrap — commonly used by men.

ค่ะ / คะ kha — commonly used by women. Use คะ for many questions.

Useful beginner sentence formula

Request + thing/action + detail + polite ending

Example: ขอกาแฟเย็น + หวานน้อย + ครับ

FunctionThaiRomanUse when...
Helloสวัสดีครับ / ค่ะsà-wàt-dii khrap / khaEntering a shop, class, condo office.
Thank youขอบคุณครับ / ค่ะkhàawp-khun khrap / khaEvery time someone helps.
Sorry / excuse meขอโทษครับ / ค่ะkhǎaw-thôot khrap / khaInterrupting or getting attention politely.
Can you speak slowly?พูดช้าๆ ได้ไหมครับ / คะphûut cháa-cháa dâai mái?When Thai comes back too fast.
I speak a little Thaiพูดไทยนิดหน่อยphûut Thai nít-nòiLower pressure and create friendliness.
Teacher cue: Correct rhythm before grammar. Learners need a successful feeling first.

Mission 1

Cafe Thai

Outcome: Order one drink naturally

Base sentence: ขอกาแฟเย็นหวานน้อยครับ / ค่ะ

khǎaw gaa-fae yen wǎan nói khrap / kha

I would like iced coffee, less sweet, please.

Phrase bank

ThaiRomanMeaning
ขอกาแฟเย็นครับ / ค่ะkhǎaw gaa-fae yen khrap / khaI would like iced coffee, please.
หวานน้อยwǎan nóiLess sweet.
ไม่ใส่น้ำตาลmâi sài náam-dtaanNo sugar.
ใส่น้ำแข็งน้อยsài náam-khǎeng nóiLess ice.
คิดเงินครับ / ค่ะkhít ngoen khrap / khaBill please.

Build your own sentence

1. Request/action
2. Item/place
3. Detail
4. Polite ending

Roleplay script

Staff: รับอะไรดีคะ? — What would you like?

Student: ขอกาแฟเย็นหวานน้อยครับ/ค่ะ

Staff: เอาหวานปกติไหม? — Normal sweet?

Student: หวานน้อยครับ/ค่ะ

ThaiRomanMeaning

Mission 6

Friendly Social Thai

Outcome: Introduce yourself simply

Base sentence: ผม/ฉันชื่อ Mike อยู่เชียงใหม่ครับ / ค่ะ

phǒm/chǎn chûue Mike, yùu Chiang Mai khrap / kha

My name is Mike. I live/stay in Chiang Mai.

Phrase bank

ThaiRomanMeaning
คุณชื่ออะไรครับ / คะkhun chûue à-rai khrap / kha?What is your name?

มาจาก...maa jàak...I come from...
