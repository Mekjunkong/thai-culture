# Static Assets

## /pdfs
Place downloadable PDF lesson sheets here.
Example: week-1-vocab-sheet.pdf

## /audio
Place native-speaker audio recordings here (MP3 recommended, max 5MB each).
Naming convention: `{lesson-slug}-{type}.mp3`
Example: week-1-greetings-male.mp3, week-1-greetings-female.mp3

## /images
Place lesson thumbnail images, cultural photos here.
Recommended: WebP format, 1280×720px for thumbnails.

Note: For large files (>10MB) consider uploading to Supabase Storage
and referencing via the lessons.audio_url / lessons.video_url columns.
